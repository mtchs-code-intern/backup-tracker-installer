# Backup Tracker Installer

A Rust-based installer for the Backup Tracker application.

## Description

It installs the most recent backup-tracker release from github, and then adds it to PATH.
## Installation

### Prerequisites

- Rust (latest stable version recommended)

### Building from Source

1. Clone the repository:
   ```bash
   git clone https://github.com/mtchs-code-intern/backup-tracker-installer.git
   cd backup-tracker-installer
   ```

2. Build the project:
   ```bash
   cargo build --release
   ```

## Usage

Best to just use the latest build in [releases](https://github.com/mtchs-code-intern/backup-tracker-installer/releases)
